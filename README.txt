 README
 
 Nagios Plugin for VMAX Reporting:
 
	- List SIDs configured on SYMCFG
	- Alerts based on Pools Status (Full and Subscription percentages).
	- Adapter Status: RF and RA adapter status.
	- Power Supply Status.
 
 Work in progress.
